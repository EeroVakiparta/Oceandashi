# Oceandashi

Small dasboard to control your digitalocean droplet(s).

### Usage
[Try it out here](https://eerovakiparta.github.io/Oceandashi/index.html)

1. Enter your authentication token
2. Control server through buttons 

### End goal is:

- Check status
- stop & start
- resize
